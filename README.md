# 📈 Trading Journal App

A simple trading journal built with **Python, Pandas and Streamlit**.

This app allows traders to:

- Create an account
- Log their trades
- Track profit and loss
- Analyze trading performance
- Visualize cumulative profit

---

## 🚀 Features

- User account system
- Personal trading history
- Profit statistics
- Cumulative profit chart
- Built with Streamlit for an easy web interface

---

## 🛠 Technologies Used

- Python
- Pandas
- Streamlit
- Matplotlib
